# 🏙️ SIMPLIFICABC

> 🚧 Em desenvolvimento (MVP)

## 📝 Sobre o Projeto
Este projeto é uma solução tecnológica voltada à melhoria do acesso à informação na comunidade de **Balneário Camboriú - SC**. A plataforma visa reunir, organizar e "traduzir" informações essenciais sobre serviços públicos (Saúde, Empregos e Assistência Social) utilizando uma linguagem clara, acessível e direta, combatendo a burocracia digital.

Projeto desenvolvido como parte do currículo acadêmico, aplicando conceitos de **Desenvolvimento Web Fullstack** e **UI/UX Design**.

---

## 🛠️ Tecnologias Utilizadas
O projeto foi construído utilizando as seguintes tecnologias:

* **Frontend:** HTML5 semântico, CSS3 (Flexbox/Grid) e JavaScript Vanilla.
* **Backend:** Node.js (em fase de implementação).
* **Design/Prototipagem:** Figma (Mockups de alta fidelidade).
* **Gestão:** Metodologia Ágil **SCRUM** via ferramenta Asana.

---

## 🎨 Diferenciais de UI/UX
Focado na **Acessibilidade Cognitiva**, o design prioriza:
* **Hierarquia Visual:** Uso de cards e ícones intuitivos para navegação rápida.
* **Linguagem Cidadã:** Substituição de termos técnicos por instruções práticas ("Onde ir", "O que levar").
* **Responsividade:** Interface adaptada para dispositivos móveis (Mobile First).

---

## 📋 Funcionalidades do MVP
1.  **💼 Central de Empregos:** Localização e requisitos para o SIME, Jovem Aprendiz e SINE.
2.  **🏥 Guia de Saúde:** Mapa de UBS e orientações sobre o Cartão Municipal.
3.  **🤝 Rede de Apoio:** Acesso rápido aos programas "Abraço" (Vida, Idoso, Mulher, Animal).
